Detailed Description

Standard implementation of downloader by UrlDownloadRequestWithIcaoCode.

This class may be used in case there is a need to download some data (represented as a string as a result) which is related to specific airport ICAO code.

Returns
Task for performing downloading and getting List<T> of type DownloadResultWithIcaoCode<S> of string as a result.
Exceptions

This materializes downloadRequests before executing main code. If it is impossible to get data (result) of any kind from a specific request from downloadRequests for any reason, this result is not added to returned collection. Before using the resulted collection, ensure it is not empty.
